# Linked Art Dataset Transformations Experimentation
Python project to develop external dataset transformations to the Linked Art Data Model
20260305

## Directories
| Folder | Description |
| -- | -- |
| output | JSON files returned by the transformation utilities |
| source | Source datasets preserved in verbatim form |
| utils | Transformation scripts organized by dataset |

- Subdirectories are organized by data source using an identifying acronym and, where applicable, the version (e.g. gadm41 = GADM Version 4.1)
